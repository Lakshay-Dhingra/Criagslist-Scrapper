About Criagslist Web Scrapper

Q. Is this website free to use?
Yes, this website is completely free to use. It is a college project which scrapes data from criagslist.org and displays it on this website.
Q. How it actually works?
First a user selects the desired city. Then the website makes request to region.criagslist.org with the desired search. For example if Delhi is selected, we will make request to delhi.criagslist.org with the search request and then scrap the important data to present to the user.
Q. Why use this website if actual website is already there?
It's your choice, both websites are non-profitable. This one uses the original website at its backend. It provides a better UI at the cost of a very slight increase in response time.
